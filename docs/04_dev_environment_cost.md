# 포크레인 배차앱 — 개발 환경 및 비용 구성표

---

## 1. 하드웨어

### 왜 Mac이 필요한가?

iOS 앱을 빌드하고 App Store에 배포하려면 **반드시 macOS + Xcode**가 필요합니다.
Flutter의 Android 빌드는 Windows에서 가능하지만, iOS 빌드는 Mac 없이 불가능합니다.

### 권장 구성

| 장비 | 모델 | 용도 | 가격 (원) |
|------|------|------|-----------|
| **MacBook Air M4 13인치** (16GB / 512GB) | 권장 | iOS+Android 빌드 + 고객 미팅 시연 | **약 189만 원** |
| MacBook Air M4 13인치 (16GB / 256GB) | 최소 사양 | 저장공간 부족 우려 | 약 159만 원 |
| MacBook Air M4 15인치 (16GB / 512GB) | 대화면 | 넓은 화면 선호 시 | 약 219만 원 |

**추천: MacBook Air M4 13인치 512GB (189만 원)**

Windows PC 없이 MacBook 하나로 iOS/Android 모두 개발 가능합니다.
고객 미팅 시 앱 시연도 가능하고, 사무실에서는 기존 모니터 2대에 USB-C로 연결하여 데스크톱처럼 사용합니다.
512GB 모델은 GPU가 10코어로 업그레이드되며, Xcode + Android Studio + Flutter를 동시 설치할 수 있는 최적 사양입니다.

### 테스트 기기

| 장비 | 용도 | 가격 |
|------|------|------|
| iOS 기기 (iPhone) | 실기기 테스트 | 보유 시 추가 비용 없음 |
| Android 기기 | 실기기 테스트 | 보유 시 추가 비용 없음 |

개인 스마트폰으로 테스트 가능합니다. 에뮬레이터도 활용할 수 있지만, 출시 전 실기기 테스트는 필수입니다.

## 2. 개발 환경 세팅

### MacBook Air (단일 개발 장비)

- Xcode → iOS 빌드
- Android Studio → Android 빌드
- Flutter SDK → 크로스플랫폼 프레임워크
- VS Code → 코드 에디팅 (주 개발 환경)
- CocoaPods → iOS 의존성 관리
- Git → 소스 관리
- Supabase 대시보드 (웹) → DB 관리

### Windows PC (기존 보유)

- 개발에 사용하지 않음
- 기존 업무/문서 작업용으로 계속 활용 가능

### 작업 흐름 (MacBook + 모니터 2대)

```
[MacBook Air 본체]               [외부 모니터 2대 (USB-C 연결)]
┌─────────────────────┐          ┌─────────────────────┐
│ 내장 화면            │          │ 모니터1: VS Code     │
│ 시뮬레이터 / 실기기  │          │ 모니터2: 터미널 +    │
│ 디버그 콘솔          │          │   Supabase 대시보드  │
└─────────────────────┘          └─────────────────────┘
```

사무실에서는 모니터 2대에 연결하여 데스크톱처럼 사용하고,
고객 미팅 시에는 MacBook만 들고 나가서 앱을 시연합니다.

## 6. 비용 절감 팁

**MacBook 대안**
- 쿠팡/중고나라에서 M2 MacBook Air 중고 구매 시 90~100만 원대 가능
- Mac mini M4(89만 원)로 대체 시 비용 절감 가능하나 고객 미팅 시연 불가

**SMS 인증 비용 절감**
- 초기에는 Supabase test OTP 사용 (무료)
- 카카오 로그인으로 전화번호 인증 대체 시 SMS 비용 제거 가능

**Supabase Free → Pro 전환 시점**
- 개발/베타 테스트 기간에는 Free 플랜 유지
- 실 사용자가 붙기 시작하면 Pro로 전환 (7일 비활성 시 프로젝트 일시정지 방지)

**카카오맵 비용**
- 카카오맵 JavaScript API(지도 표시): 무료 (트래픽 제한은 있으나 모바일 앱 WebView 사용량 기준 사실상 무제한)
- 카카오 로컬 REST API(주소→좌표 변환·키워드 검색): **일 100,000건 무료**, 초과 시 별도 요금제 협의
- Google Maps Geocoding 무료 한도(월 1만 건)보다 30배 큰 무료 한도
- 초기·확장기 모두 무료 한도 내 안전
