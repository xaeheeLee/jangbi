# 카카오맵 API 키 발급 가이드 (전중배)

> 일감 상세 지도에 카카오맵을 띄우려면 **카카오 JavaScript 키**가 필요합니다. (코드는 이미 준비됨 — 키만 넣으면 지도 표시)
> 소요 5~10분. 무료.

---

## 1. 카카오 개발자 가입·로그인
- https://developers.kakao.com 접속 → **카카오 계정으로 로그인**.
- (사업자 정보는 나중에 입력 가능 — 우선 개인 계정으로 진행 가능)

## 2. 애플리케이션 추가
- 상단 **내 애플리케이션** → **애플리케이션 추가하기**.
- **앱 이름**: `전중배`, **회사명**: 회사/개인명 입력 → 저장.

## 3. 앱 키 확인
- 생성된 앱 클릭 → 좌측 **앱 키** 메뉴.
- 여러 키가 보입니다. **우리가 쓰는 것은 `JavaScript 키`** ✅ (REST/네이티브 키는 지금 불필요)
- 이 JavaScript 키를 복사해 둡니다. (예: `a1b2c3...`)

## 4. 플랫폼 등록 (중요 — 이거 안 하면 지도가 "도메인 불일치"로 안 뜸)
- 좌측 **플랫폼** 메뉴 → **Web 플랫폼 등록** → **사이트 도메인** 추가.
  - 앱은 WebView로 지도를 띄우므로, 처음엔 아래를 등록해 두세요:
    - `http://localhost`
    - `https://localhost`
  - 실기기 테스트 중 **"지도를 표시할 수 없습니다 / 도메인 등록 필요"** 류 에러가 뜨면, **에러 메시지에 표시된 도메인을 그대로 추가**하면 됩니다.
- (선택) Android/iOS 플랫폼은 카카오 **로그인·네이티브 SDK**용 — 지도(JS)만 쓰면 **Web 등록이 핵심**입니다. 나중에 카카오 로그인 붙일 때 패키지명(`com.jangbinara.jangbinara`)·키해시·번들ID를 등록하면 됩니다.

## 5. (확인) 카카오맵 사용 가능 여부
- 보통 **JavaScript 키 + Web 도메인 등록**만으로 카카오맵 JS API가 동작합니다.
- 콘솔에 카카오맵 관련 별도 동의/활성화 항목이 있으면 켜 주세요(있을 경우).

## 6. 앱에 키 넣기
- 파일 `app/dart_define.json` 을 열어 **한 줄 추가**:
  ```json
  {
    "SUPABASE_URL": "...",
    "SUPABASE_PUBLISHABLE_KEY": "...",
    "KAKAO_JS_KEY": "여기에_복사한_JavaScript_키"
  }
  ```
  - 이 파일은 git에 올라가지 않습니다(gitignore). 예시 형식은 `app/dart_define.example.json` 참고.

## 7. 실행·확인
```bash
cd app
flutter run --dart-define-from-file=dart_define.json
```
- 일감 상세 화면 지도 자리에 **카카오맵이 실제로 표시**되면 성공.
- 안 뜨면: ① JavaScript 키 맞는지 ② Web 도메인 등록(4번) — 에러에 뜬 도메인 추가.

---

## 요약
1. developers.kakao.com → 앱 추가(전중배)
2. **앱 키 → JavaScript 키** 복사
3. **플랫폼 → Web → localhost 도메인 등록**
4. `app/dart_define.json` 에 `"KAKAO_JS_KEY": "..."` 추가
5. `flutter run --dart-define-from-file=dart_define.json` → 지도 확인

> 키 넣고 실행했는데 지도가 안 뜨면, 에러 화면/로그를 알려주세요 — 도메인 등록 등 바로잡아 드리겠습니다.
